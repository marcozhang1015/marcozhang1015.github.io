# Marco Zhang Personal Website

Static personal website designed for GitHub Pages.

## Structure

- `index.html`: two main views, with profile as the default landing page
- `styles.css`: visual system and responsive layout
- `script.js`: view switching and gallery filtering
- `assets/docs/Marco_Zhang_Resume.pdf`: downloadable CV
- `assets/images/gallery/`: photography portfolio images

## Publish on GitHub Pages

1. Create a repository such as `marcozhang1015.github.io`.
2. Copy these files into the repository root.
3. Push to GitHub.
4. In the repository settings, make sure GitHub Pages is serving from the default branch root.

## Before Publishing

- Replace `assets/images/site/profile-placeholder.svg` with your real headshot if you want the first view to show your own photo.
- Update any project descriptions or add repository links if you want each card to point somewhere specific.
